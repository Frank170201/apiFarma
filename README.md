## Ejecutar en desarrollo 

1. Clonar el repositorio
2. Ejecutar 

```bash
$ npm install
```
3. Tener Nest CLI instalado
```
$ npm install -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
